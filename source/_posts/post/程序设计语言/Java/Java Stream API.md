## Java Stream API

### 简介

Stream API 是 Java SE 8 引入的新特性。是一系列类似于 JavaScript 的函数式处理集合的方法。这种方法使我们更加关注对于集合的操作，而无需关注元素的处理顺序，这也让并行处理集合元素成为可能；

### 入门

使用步骤如下：
1. 创建流；
2. 对流做过滤、聚合、映射等操作；
3. 从流中收集元素生成集合。
