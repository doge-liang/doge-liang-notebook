## 刷题经验整理

### python

- 输入输出遇到EOFError

原因：input() 函数遇到了EOF不会自行处理
解决方法：

1. 使用 `sys.stdin`

```python
import sys
for line in sys.stdin:
    a = int(line)
    if a!= 0:
        print(a)
```

2. 使用 `try...except EOFError...`

```python
try:
    a = input()
except EOFError:
    pass
```

- 进制转换
  - 别的进制转十进制
  
  ``` Python
  int('0xA', 16) #10
  int('0o12', 8) #10
  int('1101',2) #13
  ```
  
  - 别的进制转二进制

  ``` Python
  bin(0o12) #1010
  ```
  
  - 别的进制转八进制

  ``` Python
  oct(0xA) #0o12
  ```
  
  - 别的进制转十六进制

  ``` Python
  hex(0o12) #0xA
  ```
