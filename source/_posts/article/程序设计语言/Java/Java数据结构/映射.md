## Java 的映射

## HashMap

### 初始化的两种方式

第一种方式是常见写法，实例化一个空 `HashMap` 对象，再调用实例方法 `put()`。

```JAVA
HashMap<String, String> map = new HashMap<>();
map.put("key1", "value1");
map.put("key2", "value2");
```

第二种方式使用了匿名内部类，

```JAVA
HashMap<String, String> map = new HashMap<String, String>() {
    {
        put.("key1", "value2");
        put.("key2", "value2");
    }
}
```
