插槽-slot

VUe 受 Web Components spec draft 启发实现了一个内容分配的 API 。用 <slot> 元素可以在子组件中设置一个插槽，父组件借助插槽可以定制化子组件这部分要显示的内容，具有高度的灵活性。

示例：一个简单的插槽

插槽作用域

插槽内的元素变量作用域在父组件中，因为插槽是先在父组件内生成再插入到子组件中的，所以插槽无法使用子组件内的变量。

备选内容

在 <slot> 标签内部写入内容，可以设定无外来插件的时候使用默认的内容。
