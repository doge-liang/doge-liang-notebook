C语言程序设计-基础-变量

变量

C程序的变量都必须声明后再使用

基本数据类型

  • int16, int32
  • short
  • long
  • float32
  • double
  • char8

格式输出

  • %d 十进制整数
  • %6d 按照至少6个字符宽打印整数
  • %6f 按照至少6个字符宽打印浮点数
  • %.2 按照小数点后至少2位打印浮点数

常量定义

I/O

C语言通常以流的方式读写字符；

  • c = getchar() 读入一个char
  • putchar(c) 写出一个char
  • EOF

识别文本流末尾，是一个常量标识符；
