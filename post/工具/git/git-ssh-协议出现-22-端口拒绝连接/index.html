git ssh еҚҸи®®еҮәзҺ° 22 з«ҜеҸЈжӢ’з»қиҝһжҺҘ

Posted on 2022-06-06 Edited on 2022-06-08 In е·Ҙе…· , git

git ssh еҚҸи®®еҮәзҺ° 22 з«ҜеҸЈжӢ’з»қиҝһжҺҘ

еҮәзҺ°й—®йўҳ

й—®йўҳеҲҶжһҗеҸҠи§ЈеҶі

йҖҡиҝҮ ssh е‘Ҫд»ӨзӣҙжҺҘи®ҝй—® git@github.com жЈҖжҹҘиҝһйҖҡжҖ§пјӣ

еҸ‘зҺ° github.com иў«жҳ е°„еҲ° 127.0.0.1 дәҶпјҢзҢңжөӢжҳҜејҖеҗҜдәҶз§‘еӯҰдёҠзҪ‘еҜјиҮҙзҡ„пјҢйЎәеҠҝз»ҷ git й…ҚзҪ®д»ЈзҗҶпјҢд»ҘжҸҗеҚҮжҺЁжӢүд»Јз ҒйҖҹеәҰпјӣ

еңЁ C:\User\Administrator\.ssh зӣ®еҪ•дёӢе»әз«Ӣ config ж–Үд»¶пјҢиҫ“е…ҘеҰӮдёӢеҶ…е®№пјҡ

дҝқеӯҳеҗҺеҚіеҸҜиҝһйҖҡпјӣ

зұ»дјјй—®йўҳ

з§‘еӯҰдёҠзҪ‘д№ӢеҗҺпјҢдҪҝз”Ё https зҡ„ж–№ејҸжҺЁйҖҒд»Јз Ғд№ҹж— жі•жҺЁйҖҒпјҢзҢңжөӢжҳҜеҗҢж ·зҡ„еҺҹеӣ еҜјиҮҙзҡ„гҖӮ https зҡ„д»ЈзҗҶи®ҫзҪ®е’Ң ssh дёҚеҗҢпјӣ
еҸӘиҰҒи®ҫзҪ® git config еҚіеҸҜ
