з¬¬6з« -еә”з”ЁеұӮ

еә”з”ЁеұӮдё»иҰҒеҚҸи®®

HTTP

HTTP еҚіи¶…ж–Үжң¬дј иҫ“еҚҸи®®зҡ„зј©еҶҷпјҢжҳҜи®Ўз®—жңәзҪ‘з»ңеә”з”ЁеұӮдёӯз”ЁдәҺи®Ўз®—жңәд№Ӣй—ҙдј иҫ“ж–Үеӯ—гҖҒеӣҫзүҮгҖҒйҹійў‘гҖҒи§Ҷйў‘зӯүеӨҡеӘ’дҪ“дҝЎжҒҜзҡ„зәҰе®ҡе’Ңи§„иҢғгҖӮ

еҸҜд»Ҙе·ҘдҪңеңЁе®ўжҲ·жңәе’ҢжңҚеҠЎеҷЁд№Ӣй—ҙпјҢд№ҹеҸҜд»Ҙе·ҘдҪңеңЁжңҚеҠЎеҷЁе’ҢжңҚеҠЎеҷЁд№Ӣй—ҙгҖӮ

дә”еӨ§зұ»зҠ¶жҖҒз Ғ

  • 1xxпјҢжҸҗзӨәдҝЎжҒҜпјҢиЎЁзӨәзӣ®еүҚеӨ„дәҺеҚҸи®®еӨ„зҗҶзҡ„дёӯй—ҙзҠ¶жҖҒпјҢиҝҳйңҖиҰҒеҗҺз»ӯж“ҚдҪңпјӣ
  • 2xxпјҢжҲҗеҠҹпјҢжҠҘж–ҮеҲ°иҫҫе№¶жӯЈзЎ®еӨ„зҗҶпјӣпјҲ200гҖҒ204гҖҒ206пјү
    • [200 OK] пјҡеҰӮжһңжҳҜйқһ HEAD иҜ·жұӮдёҖиҲ¬йғҪдјҡжңү body ж•°жҚ®пјӣ
    • [204 No Content] пјҡжІЎжңү body зҡ„жҲҗеҠҹиҜ·жұӮпјӣ
    • [206 Partial Content] пјҡдё»иҰҒеә”з”ЁдәҺ HTTP еҲҶеқ—жҲ–иҖ…ж–ӯзӮ№з»ӯдј пјҢиЎЁзӨәиҝ”еӣһзҡ„ж•°жҚ®еҸӘжҳҜдёҖйғЁеҲҶпјҢе№¶дёҚе…Ёпјӣ
  • 3xxпјҢйҮҚе®ҡеҗ‘пјҢиө„жәҗдҪҚзҪ®жңүеҸҳеҠЁпјҢйңҖиҰҒе®ўжҲ·жңәйҮҚж–°еҸ‘йҖҒиҜ·жұӮпјӣпјҲ301гҖҒ302гҖҒ304пјү
    • [301 Moved Permanently] пјҡж°ёд№…йҮҚе®ҡеҗ‘пјҢиө„жәҗе·ІдёҚеӯҳеңЁпјҢйңҖиҰҒе®ўжҲ·з«ҜдҪҝз”Ёж–°зҡ„ url иҜ·жұӮпјӣ
    • [302 Found] пјҡдёҙж—¶йҮҚе®ҡеҗ‘пјҢиЎЁзӨәиө„жәҗиҝҳеңЁпјҢдҪҶдёҙж—¶йңҖиҰҒз”ЁеҲ«зҡ„ url иҜ·жұӮпјӣ

      301 е’Ң 302 йғҪдјҡеңЁ Location еӯ—ж®өжҢҮжҳҺйҮҚе®ҡеҗ‘зҡ„ж–° url жөҸи§ҲеҷЁдјҡиҮӘеҠЁи·іиҪ¬пјӣ

    • [304 Not Modified] пјҡдёҚе…·жңүи·іиҪ¬зҡ„еҗ«д№үпјҢиЎЁзӨәиө„жәҗжңӘдҝ®ж”№пјҢйҮҚе®ҡеҗ‘е·ІеӯҳеңЁзҡ„зј“еҶІж–Үд»¶пјҢд№ҹз§°зј“еӯҳйҮҚе®ҡеҗ‘пјҢз”ЁдәҺзј“еӯҳжҺ§еҲ¶пјӣ
  • 4xxпјҢе®ўжҲ·з«Ҝй”ҷиҜҜпјҢиҜ·жұӮжңүиҜҜпјҢжңҚеҠЎеҷЁж— жі•еӨ„зҗҶпјӣпјҲ400гҖҒ403гҖҒ404пјү
    • [400 Bad Request] пјҡиҜ·жұӮжҠҘж–ҮжңүиҜҜпјӣпјҲдёҚзҹҘйҒ“е“ӘйҮҢжңүиҜҜпјҢеҸҚжӯЈжңүиҜҜпјү
    • [403 Forbidden] пјҡжңҚеҠЎеҷЁзҰҒжӯўи®ҝй—®иҜҘиө„жәҗпјӣ
    • [404 Not Found] пјҡиҜ·жұӮиө„жәҗеңЁжңҚеҠЎеҷЁжңӘжүҫеҲ°пјӣ
  • 5xxпјҢжңҚеҠЎз«Ҝй”ҷиҜҜпјҢжңҚеҠЎеҷЁеңЁеӨ„зҗҶиҜ·жұӮж—¶еҸ‘з”ҹеҶ…йғЁй”ҷиҜҜпјӣпјҲ500гҖҒ501гҖҒ502гҖҒ503пјү
    • [500 Internal Server Error] пјҡиЎЁзӨәжңҚеҠЎеҷЁеҸ‘з”ҹй”ҷиҜҜпјӣпјҲдёҚзҹҘйҒ“е“ӘйҮҢй”ҷдәҶпјҢеҸҚжӯЈжҳҜй”ҷдәҶпјү
    • [501 Not Implemented] пјҡиЎЁзӨәжүҖиҜ·жұӮзҡ„еҠҹиғҪе°ҡжңӘе®һзҺ°пјӣпјҲе°ҪиҜ·жңҹеҫ…пјү
    • [502 Bad Gateway] пјҡжңҚеҠЎеҷЁдҪңдёәзҪ‘е…іжҲ–д»ЈзҗҶж—¶иҝ”еӣһзҡ„й”ҷиҜҜз ҒпјҢиЎЁзӨәжңҚеҠЎеҷЁиҮӘиә«е·ҘдҪңжӯЈеёёпјҢи®ҝй—®еҗҺз«ҜжңҚеҠЎеҷЁеҸ‘з”ҹдәҶй”ҷиҜҜпјӣпјҲеҗҺз«Ҝи®ҝй—®еҗҺз«Ҝзҡ„еҗҺз«Ҝж—¶еҸ‘з”ҹй”ҷиҜҜпјү
    • [503 Service Unavailable] пјҡжңҚеҠЎеҷЁеҝҷпјҢзЁҚеҗҺеҶҚиҜ•пјӣ

жҠҘж–Үеӯ—ж®ө

  • Host пјҡжҢҮе®ҡи®ҝй—®еҹҹеҗҚпјӣ
  • Content-Length пјҡеӣһеә”зҡ„жҠҘж–Үй•ҝеәҰпјӣ
  • Connection пјҡ
