特征值和特征向量

定义：特征向量、特征多项式
A\bm{A}A 是一个 nnn 阶矩阵， λ\lambdaλ 是一个数，如果存在非零向量 α\bm{\alpha}α 使得：

Aα=λα\bm{A\alpha} = \lambda \bm{\alpha} Aα=λα

就称 λ\lambdaλ 是 A\bm{A}A 的一个特征值， α\bm{\alpha}α 是 A\bm{A}A 属于 λ\lambdaλ 的一个特征向量。

定义：特征矩阵、特征多项式
设 A=(aij)nn\bm{A} = (a_{ij})_{nn}A=(aij​)nn​ 为数域 FFF 上的 nnn 阶方阵，我们称关于 λ\lambdaλ 的多项式 f(λ)=∣λE−A∣f(\lambda) = |\lambda \bm{E} - \bm{A}|f(λ)=∣λE−A∣ 为矩阵 A\bm{A}A 的特征多项式，称 λE−A\lambda \bm{E} - \bm{A}λE−A 为矩阵 A\bm{A}A 的特征矩阵。

求特征值和特征向量的方法：

  1. 求解 ∣λE−A∣=0|\lambda \bm{E} - \bm{A}| = 0∣λE−A∣=0 算出所有的根 λi\lambda_iλi​；
  2. 对每个 λi\lambda_iλi​ 计算出所有的 (λiE−A)α=0(\lambda_i \bm{E} - \bm{A})\bm{\alpha} = 0(λi​E−A)α=0 的基础解系，得到属于每一个 λi\lambda_iλi​ 的特征向量 α\bm{\alpha}α

特征值和特征向量的一些性质：

  1. ∑i=1nλi=∑i=1naii=tr(A)\sum \limits_{i = 1}^{n}\lambda_i = \sum \limits_{i = 1}^{n}a_{ii} = \mathrm{tr}(\bm{A})i=1∑n​λi​=i=1∑n​aii​=tr(A);
  2. ∑i=0nλi=∣A∣\sum \limits_{i = 0}^{n}\lambda_i = |\bm{A}|i=0∑n​λi​=∣A∣;
  3. 对矩阵 A\bm{A}A 的所有特征值 λi\lambda_iλi​ 有属于 λi\lambda_iλi​ 的特征向量组 {αik∣k∈[1,t]}\{\bm{\alpha}_{ik} \vert k \in [1, t] \}{αik​∣k∈[1,t]} 线性无关，那么所有的 {αik∣k∈[1,t],i∈[1,s]}\{\bm{\alpha}_{ik} \vert k \in [1, t], i \in [1, s] \}{αik​∣k∈[1,t],i∈[1,s]} 都线性无关;

相似矩阵和矩阵对角化

定义：相似矩阵
对于矩阵 A\bm{A}A 存在可逆矩阵 P\bm{P}P 使得 P−1AP=B\bm{P}^{-1}\bm{AP} = \bm{B}P−1AP=B 我们称之为 A\bm{A}A 相似于 B\bm{B}B ，记为 A∼B\bm{A} \sim \bm{B}A∼B .

矩阵间的相似有以下性质：

  1. 反身性
  2. 对称性
  3. 传递性

此外还有一些特殊的性质：

  1. ∣A∣=∣B∣|\bm{A}| = |\bm{B}|∣A∣=∣B∣
  2. A\bm{A}A 和 B\bm{B}B 拥有相同的特征多项式和特征值；
  3. tr(A)=tr(B)\mathrm{tr}(\bm{A}) = \mathrm{tr}(\bm{B})tr(A)=tr(B) 且有 r(A)=r(B)r(\bm{A}) = r(\bm{B})r(A)=r(B);
  4. 如果 f(x)f(x)f(x) 是一个多项式，那么 f(B)=P−1f(A)Pf(\bm{B}) = \bm{P}^{-1} f(\bm{A}) \bm{P}f(B)=P−1f(A)P
  5. AT∼BT\bm{A}^T \sim \bm{B}^TAT∼BT ，若 A\bm{A}A 可逆还有 A∗∼B∗\bm{A}^*\sim\bm{B}^*A∗∼B∗

矩阵对角化就是求一个矩阵 P\bm{P}P 使得矩阵 A\bm{A}A 和一个对角矩阵 Λ\varLambdaΛ 相似。
矩阵可对角化的充要条件：矩阵 A\bm{A}A 有 nnn 个线性无关的特征向量。
推论 1：一个 nnn 阶矩阵 A\bm{A}A 有 nnn 个不同的特征值，则 A\bm{A}A 可对角化。
推论 2：一个 nnn 阶复矩阵 A\bm{A}A 的特征多项式无重根，则 A\bm{A}A 可对角化。

定义：内积空间、长度、夹角、正交
定义向量的内积：有列向量 α\bm{\alpha}α 和列向量 β\bm{\beta}β 我们将 (α,β)=αTβ=x1y1+x2y2+⋯+xnyn(\bm{\alpha, \beta}) = \bm{\alpha}^T\bm{\beta} = x_1y_1 + x_2y_2 + \cdots + x_ny_n(α,β)=αTβ=x1​y1​+x2​y2​+⋯+xn​yn​ 称为向量 α\bm{\alpha}α 和向量 β\bm{\beta}β 的内积。定义了线性运算和内积运算的向量空间 Rn\R^nRn 叫内积空间。

内积空间内的向量与它本身的内积的开平方，即 ∣α∣=(α,α)=x12+x22+⋯+xn2|\bm{\alpha}| = \sqrt{(\bm{\alpha}, \bm{\alpha})} = \sqrt{x_{1}^{2} + x_{2}^{2} + \cdots + x_{n}^{2}}∣α∣=(α,α)​=x12​+x22​+⋯+xn2​​ 作为向量 α\bm{\alpha}α 的距离。

有了距离和内积的定义，我们就有了夹角的定义， cos⁡θ=(α,β)∣α∣⋅∣β∣\cos \theta = \dfrac{(\bm{\alpha}, \bm{\beta})}{|\bm{\alpha}| \cdot |\bm{\beta}|}cosθ=∣α∣⋅∣β∣(α,β)​ ，我们将 θ\thetaθ 定义为两个向量的夹角，记为 ⟨α,β⟩\lang \bm{\alpha}, \bm{\beta} \rang⟨α,β⟩。

当夹角 θ=π2\theta = \dfrac{\pi}{2}θ=2π​ 即 cos⁡θ=0\cos\theta = 0cosθ=0 时，我们将这种情况称为正交。一组两两正交的非零向量构成的向量组称为正交向量组。一组正交向量组若构成了内积空间 Rn\R^nRn 的一组基，那么这组向量称为正交基。如果都是单位向量，那么被称为标准正交基。

根据以上的定义，我们有以下的不等式：
柯西不等式：

∣(α,β)∣⩽∣α∣⋅∣β∣|(\bm{\alpha}, \bm{\beta})| \leqslant |\bm{\alpha}| \cdot |\bm{\beta}| ∣(α,β)∣⩽∣α∣⋅∣β∣

三角不等式：

∣α∣+∣β∣⩾∣α∣+∣β∣|\bm{\alpha}| + |\bm{\beta}| \geqslant |\bm{\alpha}| + |\bm{\beta}| ∣α∣+∣β∣⩾∣α∣+∣β∣

定理 1：
正交向量组是线性无关向量组。

定理 2：
（施密特正交化）通过已知的一组基构造出一组正交基。

定义：正交矩阵
若矩阵 A\bm{A}A 满足 ATA=E\bm{A}^T \bm{A} = \bm{E}ATA=E 则称其为正交矩阵。
性质：

  1. A\bm{A}A 可逆，那么 A−1=AT\bm{A}^{-1} = \bm{A}^TA−1=AT；
  2. A−1,AT\bm{A}^{-1}, \bm{A}^TA−1,AT 也是正交矩阵；
  3. ∣A∣=±1|\bm{A}| = \pm 1∣A∣=±1；
  4. 正交矩阵的乘积依然是正交矩阵；

定理 3：
矩阵 A=(α1,α2,⋯ ,αn)\bm{A} = (\bm{\alpha}_1, \bm{\alpha}_2, \cdots, \bm{\alpha}_n)A=(α1​,α2​,⋯,αn​) 是正交矩阵的充要条件是列向量 α1,α2,⋯ ,αn\bm{\alpha}_1, \bm{\alpha}_2, \cdots, \bm{\alpha}_nα1​,α2​,⋯,αn​ 构成矩阵的一组标准正交基。

定理 4：
实对称矩阵的特征值都是实数。

定理 5：
实对称矩阵属于不同特征值的特征向量必定正交。

定理 6：
对于实对称矩阵 A\bm{A}A 存在正交矩阵 T\bm{T}T 使得 T−1AT\bm{T}^{-1} \bm{A} \bm{T}T−1AT 为对角矩阵。

如何求 T\bm{T}T ？：

  1. 求出矩阵 A\bm{A}A 的所有特征值 λi\lambda_iλi​ ，以及属于他们的特征向量 α1si,α2si,⋯ ,αisi\bm{\alpha}_{1s_i}, \bm{\alpha}_{2s_i}, \cdots, \bm{\alpha}_{is_i}α1si​​,α2si​​,⋯,αisi​​；
  2. 将属于 λi\lambda_iλi​ 的特征向量 α1si,α2si,⋯ ,αisi\bm{\alpha}_{1s_i}, \bm{\alpha}_{2s_i}, \cdots, \bm{\alpha}_{is_i}α1si​​,α2si​​,⋯,αisi​​ 正交化和单位化；
  3. 将上一步处理后的所有特征向量合并，得到的向量组就是单位正交向量组。
