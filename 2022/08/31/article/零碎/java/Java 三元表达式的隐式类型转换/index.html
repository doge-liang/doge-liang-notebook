Java 三元表达式的隐式类型转换

Posted on 2022-08-31 Edited on 2025-02-01 In article , 零碎 , java

Java 三元表达式的隐式类型转换

  1. 如果定义了数据类型的变量与未定义变量的数值共同参与三元运算符的后双目运算，那么返回的结果就是**范围大（精度高）**类型；
  2. 如果两个定义了数据类型的变量共同参与三元运算符的后双目运算，那么返回的结果就是**范围大（精度高）**类型；
  3. 如果直接进行数值的比较，会自动转型成为范**围大（精度高）**的数据类型；
